A [Deckset](https://www.deckset.com/) theme of the GitHub presentation template is available for download for the members of the GitHub organization on [Google Drive](https://drive.google.com/open?id=1PDKiK_hOJja6ZBQiTHy37JD8S4Buzwz7).

![deckset-template](https://user-images.githubusercontent.com/10384315/77022093-e8f12880-6945-11ea-92f9-623556fa6488.png)

An example presentation is also provided to showcase various slide options when using the theme but please refer to the official [Deckset help site](https://docs.decksetapp.com/English.lproj/getting-started.html) for more detailed information.

### How to use

1. Download the assets
2. Open Deckset and click **File > Import Theme...**
3. Navigate to the `GitHub Presentation Template - Deckset` folder and select `GitHub_Theme.dstheme`

#### Basic customization

Here is a basic set of commands that you can add to individual slides to customize your content:

|              Command              |                        Description                        |
| :-------------------------------: | :-------------------------------------------------------: |
| `[.background-color: #HEXVALUE]`  |   Changes the background color for an individual slide    |
|         `[.hide-footer]`          |         Hides the footer for an individual slide          |
|   `[.slidenumbers:true/false]`    | Shows or hides the slide numbers for an individual slide. |
| `[.code-highlight: LINE NUMBERS]` |          Step through highlighted lines of code.          |

Learn more about customizing your slides on the [Deckset help site](https://docs.decksetapp.com/English.lproj/getting-started.html).
