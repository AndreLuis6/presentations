The GitHub presentation template is available as a Figma library for members of the GitHub organization.

### How to use

To use:

1. Open `GitHub Presentation Template` located in the Primer team in Figma.
2. Duplicate the file into your own drafts
3. Open the duplicated file and configure your own presentation

- Tip: Don't forget to rename the duplicated presentation file.

For more information on how to duplicate files in Figma, please refer to [Figma's documentation](https://help.figma.com/hc/en-us/articles/360038511533-Duplicate-a-File#duplicate).
